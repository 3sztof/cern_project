NAME

       fmcSrvViewer - Print the execution status (on/off) of the FMC Servers

SYNOPSIS

       fmcSrvViewer [-N DIM_DNS_NODE] [-m NODE_PATTERN...]  [-C] [-M]

       fmcSrvViewer -h

DESCRIPTION

       The  command  fmcSrvViewer(1)  is a FMC debugging tool which prints the
       execution status (on/off) of the FMC Servers which are supposed  to  be
       running on the farm nodes registered with the DIM Name Server in execu-
       tion on the node DIM_DNS_NODE.

       FMC servers are classified in  Control  Servers  [logSrv(8),  tmSrv(8),
       pcSrv(8), ipmiSrv(8), cmSrv(8)] and Monitor Servers [coalSrv(8), cpuin-
       foSrv(8), cpustatSrv(8),  fsSrv(8),  irqSrv(8),  memSrv(8),  nifSrv(8),
       osSrv(8), psSrv(8), smartSrv(8) and tcpipSrv(8)].

       The  DIM  Name  Server DIM_DNS_NODE, looked up to seek for FMC Servers,
       can be chosen (in decreasing order of priority) by:

              1. Specifying the -N DIM_DNS_NODE command-line option;

              2. Specifying the DIM_DNS_NODE environment variable;

              3. Defining the DIM_DNS_NODE in the file "/etc/sysconfig/dim".

OPTIONS

       -h     Get basic usage help from the command line, then terminate.

       -N DIM_DNS_NODE (string)
              Use DIM_DNS_NODE as DIM Name Server node.

       -m NODE_PATTERN (string, repeatable)
              Print only the status of the servers running on the nodes  whose
              hostname   matches   at  least  one  of  the  wildcard  patterns
              NODE_PATTERN (default: print the status of the  servers  running
              on   all   the   nodes  registered  with  the  DIM  Name  Server
              DIM_DNS_NODE).

       -C     Print  only  the  status  of  the  Control  Servers  [logSrv(8),
              tmSrv(8),  pcSrv(8),  ipmiSrv(8),  cmSrv(8)]. Default: print the
              status all the FMC servers.

       -M     Print only the status of the Monitor Servers [coalSrv(8), cpuin-
              foSrv(8),  cpustatSrv(8),  fsSrv(8),  irqSrv(8), memSrv(8), nif-
              Srv(8),  osSrv(8),  psSrv(8),  smartSrv(8)   and   tcpipSrv(8)].
              Default: print the status all the FMC servers.

ENVIRONMENT

       DIM_DNS_NODE (string, mandatory if not defined otherwise, see above)
              Host name of the node which is running the DIM DNS.

       LD_LIBRARY_PATH (string, mandatory if not set using ldconfig)
              Must  include  the  path to the libraries "libdim" and "libFMCu-
              tils".

EXAMPLES

       Print the status of all the FMC servers registered with  the  DIM  Name
       Server DIM_DNS_NODE:

       bash$ fmcSrvViewer

       Print the status of all the FMC control servers registered with the DIM
       Name Server DIM_DNS_NODE:

       bash$ fmcSrvViewer -C

       Print the status of all the FMC monitor servers registered with the DIM
       Name Server DIM_DNS_NODE:

       bash$ fmcSrvViewer -M

       Print  the  status of the FMC control servers running on nodes farm0101
       and farm0102:

       bash$ fmcSrvViewer -m farm0101 -m farm0102 -C

       Use wildcard patterns to select the nodes:

       bash$ fmcSrvViewer -m "farm01??" -m "farm02??" -C
       bash$ fmcSrvViewer -m "farm010[1-3]"
