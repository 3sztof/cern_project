NAME

       osViewer - Get the Operating System information about the farm PCs from
                  the FMC Operating System Information Servers

SYNOPSIS

       osViewer [-v | --debug] [-N | --dim-dns DIM_DNS_NODE]
                [-m | --hostname NODE_PATTERN...]  [-c | --compact]
                [-H | --header]

       osViewer { -V | --version } [-v | --debug]
                [-N | --dim-dns DIM_DNS_NODE]
                [-m | --hostname NODE_PATTERN...]

       osViewer { -h | --help }

DESCRIPTION

       Get the  Operating  System  information  about  the  farm  nodes  whose
       hostname matches the wildcard pattern NODE_PATTERN, specified using the
       -m NODE_PATTERN command line option (default: all the nodes  registered
       with the DIM name server DIM_DNS_NODE). Information includes:

       Name (string)
              The OS name (e.g.: "Linux").

       Distribution (string)
              The  OS  distribution  (e.g.: "Scientific Linux CERN SLC release
              4.6 (Beryllium)").

       Kernel (string)
              The OS kernel (e.g.: "2.6.9-55.0.12.EL.cernsmp").

       Release (string)
              The OS release (e.g.: "#1 SMP Mon Nov 5 17:20:17 CET 2007").

       Machine (string)
              The machine architecture (e.g.: "x86_64", "i686").

       LastBootUpTime (string)
              The date and time of the last reboot (e.g.:  "2012  Apr  19  Thu
              09:20:09 CEST").

       LocalDateTime (string)
              The  local  date  and  time  of the node (e.g.: "2012 May 08 Tue
              16:38:29 CEST").

       The DIM Name Server DIM_DNS_NODE, looked up to seek for FMC OS Informa-
       tion Servers, can be chosen (in decreasing order of priority) by:

              1. Specifying the -N DIM_DNS_NODE command-line option;

              2. Specifying the DIM_DNS_NODE environment variable;

              3. Defining the DIM_DNS_NODE in the file "/etc/sysconfig/dim".

OPTIONS

       -h --help
              Get basic usage help from the command line, then terminate.

       -v --debug
              Increase verbosity for debugging.

       -N --dim-dns DIM_DNS_NODE (string)
              Use DIM_DNS_NODE as DIM Name Server node.

       -m --hostname NODE_PATTERN (string, repeatable)
              Print  only  the  Operating System information of the farm nodes
              whose hostname matches at least one  of  the  wildcard  patterns
              NODE_PATTERN (default: print the Operating System information of
              all the nodes registered with the DIM name server DIM_DNS_NODE).

       -V --version
              For  each selected node print the FMC OS Information Server ver-
              sion and the FMC version, than terminate.

       -c --compact
              Print the Operating System information in compact format (easier
              to  parse):  one  farm  node  per line, no labels, spaces inside
              fields replaced by underscores.

       -H --header
              If the compact output format has been choosen, print  a  heading
              line with the output field labels.

ENVIRONMENT

       DIM_DNS_NODE (string, mandatory if not defined otherwise, see above)
              Host name of the node which is running the DIM DNS.

       LD_LIBRARY_PATH (string, mandatory if not set using ldconfig)
              Must  include  the  path to the libraries "libdim" and "libFMCu-
              tils".

EXAMPLES

       Print the Operating System information of all the farm nodes:

       bash$ osViewer

       Print output in compact format with header:

       bash$ osViewer -c -H
       bash$ osViewer -cH
       bash$ osViewer --comp --hea

       Print only the Operating System information of selected nodes:

       bash$ osViewer -m farm0101 -m farm0102
       bash$ osViewer -host farm0101 -host farm0102

       Print only the Operating System information of wildcard-selected nodes:

       bash$ osViewer -m "farm01*" -m "farm02*"
       bash$ osViewer -m "farm01?[1357]"
       bash$ osViewer -m "farm01?[1-5]"
       bash$ osViewer --host "farm01[0-5][1-36-9]"

       Print the Server version and the FMC package version of the running FMC
       Operating System Information servers registered with  the  current  DIM
       name server:

       bash$ osViewer -V
