NAME

       fmcVersionViewer - Print the revision tag of the main source file and
                          the version tag of the FMC package of the FMC
                          Servers currently running

SYNOPSIS

       fmcVersionViewer [-v...]  [-N DIM_DNS_NODE] [-m NODE_PATTERN...]  [-C]
                        [-M] [-s SERVER_NAME...]

       fmcVersionViewer -h

DESCRIPTION

       The command fmcVersionViewer(1) is a FMC debugging tool which prints to
       the  standard  output  stream the revision tag (RCS header) of the main
       source file and the version tag of the FMC package of the  FMC  servers
       registered with the DIM Name Server running on the node DIM_DNS_NODE.

       Contacted  FMC  servers are: logSrv(8), tmSrv(8), pcSrv(8), ipmiSrv(8),
       cmSrv(8),   coalSrv(8),   cpuinfoSrv(8),    cpustatSrv(8),    fsSrv(8),
       irqSrv(8),   memSrv(8),  nifSrv(8), osSrv(8), psSrv(8), smartSrv(8) and
       tcpipSrv(8).

       The DIM Name Server DIM_DNS_NODE, looked up to seek  for  FMC  Servers,
       can be chosen (in decreasing order of priority) by:

              1. Specifying the -N DIM_DNS_NODE command-line option;

              2. Specifying the DIM_DNS_NODE environment variable;

              3. Defining the DIM_DNS_NODE in the file "/etc/sysconfig/dim".

OPTIONS

       -h     Get basic usage help from the command line, then terminate.

       -v     Increase verbosity for debugging.

       -N DIM_DNS_NODE (string)
              Use DIM_DNS_NODE as DIM Name Server node.

       -m NODE_PATTERN (string, repeatable)
              Print only the version of the servers running on the nodes whose
              hostname  matches  at  least  one  of  the   wildcard   patterns
              NODE_PATTERN  (default: print the version of the servers running
              on  all  the  nodes  registered  with  the   DIM   Name   Server
              DIM_DNS_NODE).

       -s SERVER_NAME (string, repeatable)
              Print  only  the  version  of  server SERVER_NAME. More than one
              server can be specified by repeating the -s option. Default  (if
              no  option  -s  is  specified): print the version of all the FMC
              servers. Available SERVER_NAMEs are:

                     log     FMC Message Logger Server;
                     tm      FMC Task Manager Server;
                     pc      FMC Process Controller Server;
                     ipmi    FMC Power Manager Server;
                     cm      FMC Configuration Manager Server;
                     coal    FMC Network Inteface Coalescence Monitor Server;
                     cpuinfo FMC CPU Static Information Monitor Server;
                     cpustat FMC CPU state information Monitor Server;
                     fs      FMC File System Monitor Server;
                     irq     FMC Interrupt Monitor Server;
                     mem     FMC Memory Monitor Server;
                     nif     FMC Network Interface Monitor Server;
                     os      FMC Operating System Monitor Server;
                     ps      FMC Process Monitor Server;
                     smart   FMC Disk health Monitor Server;
                     tcpip   FMC TCP/IP Stack Monitor Server.

       -C     Print only the version of the Control Servers. Equivalent to "-s
              log -s tm -s pc -s ipmi -s cm".

       -M     Print only the version of the Monitor Servers. Equivalent to "-s
              coal -s cpuinfo -s cpustat -s fs -s irq -s mem -s nif -s  os  -s
              ps -s smart -s tcpip".

ENVIRONMENT

       DIM_DNS_NODE (string, mandatory if not defined otherwise, see above)
              Host name of the node which is running the DIM DNS.

       LD_LIBRARY_PATH (string, mandatory if not set using ldconfig)
              Must  include  the  path to the libraries "libdim" and "libFMCu-
              tils".

EXAMPLES

       Print the version of all the FMC servers registered with the  DIM  Name
       Server DIM_DNS_NODE:

       bash$ fmcVersionViewer

       Print  the  version  of all the FMC control servers registered with the
       DIM Name Server DIM_DNS_NODE:

       bash$ fmcVersionViewer -C

       Print the version of the FMC servers Task Manager servers and  Configu-
       ration Manager servers running on nodes farm0101 and farm0102:

       bash$ fmcVersionViewer -m farm0101 -m farm0102 -s tm -s cm

       Use wildcard patterns to select the nodes:

       bash$ fmcVersionViewer -m "farm01??" -m "farm02??" -s tm -s cm
       bash$ fmcVersionViewer -m "farm010[1-3]"
